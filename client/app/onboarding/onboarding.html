Onboarding

Setting up tracking for your website is as easy as 1-2-3! Just follow these simple steps:

1. Register Your Website

Register the site you wish to track by providing the website's name and url in the boxes below. When designating url, please type the part of your website address that comes after "http://" or "https://". Remember, if your website has different pages, only include the basic url without any redirects.


Correct format:
  • sandpiperanalytics.herokuapp.com
Incorrect format:
  • sandpiperanalytics
  • http://sandpiperanalytics.herokuapp.com
  • sandpiperanalytics.herokuapp.com/
  • sandpiperanalytics.herokuapp.com/dashboard

2. Insert code

Add the following code to the "head" section of your webpage: 

{{script}}
Note: If your website is a single-page application, you only need to add this code to one file, usually titled "index.html". If you have a multi-page application, you'll need to include include this script in the "head" section of each page that you want to track.
